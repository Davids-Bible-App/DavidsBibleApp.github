## Bible Translations (.dba SQLite)

This repository hosts Bible translations optimized for **Davids Bible App**. These files use the `.dba` extension but are standard SQLite databases.

## 🛠 How to Use

1. **Individual Files**: Click "Download" above for specific translations.
2. **Offline Backup**: Download the entire `Translations` folder to keep a local copy of all databases and cross-references.
3. **Installation**: Use the app's **Upload** feature to import the `.dba` files. They will be available instantly.
   - _Note: KJV Strongs data may require an app restart or refresh to initialize._
