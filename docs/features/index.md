# Features

## 📖 Clean Reading Experience

A distraction‑free layout designed for long reading sessions.

## 🔍 Powerful Search

Search by:

- Keyword
- Phrase
- Book
- Translation

Results appear instantly.

## 📚 Parallel Translations

Compare verses side‑by‑side across multiple translations.

## 🎨 Themes

Choose between:

- Light
- Dark
- Sepia

## 📌 Bookmarks & Highlights

Save verses for later and organize your study.

## 🌐 Offline Support

All translations are stored locally — no internet required.

## ⚙️ Custom Translation Files

Add your own translation files or update existing ones.

See the [Help Center](/help/) for detailed usage.
